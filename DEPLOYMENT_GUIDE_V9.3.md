## 🔧 إدارة البيانات:

### إضافة باقة جديدة:

**الطريقة 1: من Admin Panel**
1. افتح https://dolcera.me?admin
2. سجل دخول
3. اذهب لـ "المنتجات"
4. اضغط "+ إضافة باقة"
5. املأ البيانات
6. احفظ

**الطريقة 2: من Supabase مباشرة**
```sql
INSERT INTO products (
  name, base_price, stock_quantity, images, contents
) VALUES (
  'باقة جديدة',
  1000,
  5,
  '["https://example.com/image.jpg"]'::jsonb,
  '["محتوى 1", "محتوى 2"]'::jsonb
);
```

### تعديل باقة موجودة:

**من Admin Panel:**
1. اذهب لـ "المنتجات"
2. اضغط "تعديل" على الباقة
3. عدّل البيانات
4. احفظ

**من Supabase:**
```sql
UPDATE products 
SET base_price = 1600 
WHERE name = 'باقة رويال الفاخرة';
```

### حذف باقة:

**من Admin Panel:**
1. اذهب لـ "المنتجات"
2. اضغط "حذف" على الباقة
3. أكد الحذف

## 🧪 اختبار نموذج الحجز:

### خطوات الاختبار:
1. افتح https://dolcera.me
2. اضغط "تصفح الباقات"
3. اختر باقة
4. اضغط "احجز الآن"
5. **الخطوة 1:** اختر التاريخ
6. **الخطوة 2:** حدد المدة
7. **الخطوة 3:** املأ البيانات:
   - الاسم
   - الجوال
   - **البريد الإلكتروني** (جديد)
   - **المدينة** (جديد - يظهر سعر التوصيل تلقائياً)
   - العنوان
8. **الخطوة 4:** راجع الملخص واختر طريقة الدفع
9. اضغط "تأكيد الحجز"
10. **النتيجة:** صفحة تأكيد مع زر واتساب

### ما يجب أن تراه:
✅ قائمة المدن تظهر
✅ عند اختيار مدينة، يظهر: "📦 سعر التوصيل: XX ر.س"
✅ الإجمالي يتضمن سعر التوصيل
✅ البريد الإلكتروني إلزامي
✅ زر "تأكيد عبر واتساب" يفتح رسالة جاهزة

## 🔍 استكشاف الأخطاء:

### المشكلة: الباقات لا تزال لا تظهر

**الحل:**
1. افتح Console (F12)
2. اكتب: `console.log(D.products)`
3. إذا رأيت `[]` فارغ:
   ```javascript
   // امسح localStorage وحدّث الصفحة
   localStorage.clear();
   location.reload();
   ```

### المشكلة: قائمة المدن فارغة

**الحل:**
1. افتح Console
2. اكتب: `console.log(D.locations)`
3. إذا فارغ:
   ```javascript
   localStorage.clear();
   location.reload();
   ```

### المشكلة: الحجز لا يرسل لـ Supabase

**التحقق:**
1. افتح Supabase Dashboard
2. اذهب لـ Table Editor > bookings
3. تحقق من آخر حجز

**الحل:**
1. تأكد من RLS Policies:
   ```sql
   -- في SQL Editor
   CREATE POLICY "Anyone can create bookings" ON bookings
     FOR INSERT WITH CHECK (true);
   ```

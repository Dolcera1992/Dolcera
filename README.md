## 🌟 المميزات

### للعملاء:
- ✅ **تصفح الباقات** - عرض جميع طاولات الضيافة المتاحة
- ✅ **نظام حجز ذكي** - كشف تلقائي لتعارض المواعيد
- ✅ **حساب تلقائي** - الأسعار + الضريبة + التوصيل + التأمين
- ✅ **اختيار المدينة** - أسعار توصيل ديناميكية حسب المنطقة
- ✅ **تسجيل الدخول** - Google OAuth أو البريد الإلكتروني
- ✅ **حجوزاتي** - متابعة جميع الحجوزات السابقة
- ✅ **تتبع الطلب** - معرفة حالة الحجز في الوقت الفعلي
- ✅ **واتساب مباشر** - تأكيد الحجز عبر رسالة جاهزة

### للإدارة:
- ✅ **لوحة تحكم شاملة** - Dashboard كامل للإدارة
- ✅ **إدارة المنتجات** - إضافة/تعديل/حذف الباقات
- ✅ **إدارة الحجوزات** - تحديث الحالة، عرض التفاصيل
- ✅ **إدارة المدن** - إضافة مدن جديدة وأسعار التوصيل
- ✅ **نظام الإشعارات** - تنبيهات للحجوزات الجديدة
- ✅ **الكوبونات** - إنشاء أكواد خصم نسبية أو ثابتة
- ✅ **الفواتير** - طباعة فواتير PDF للعملاء
- ✅ **التقارير** - إحصائيات الإيرادات والحجوزات
- ✅ **تخصيص الموقع** - 5 ثيمات ألوان + تعديل النصوص

## 🛠️ التقنيات المستخدمة

| المجال | التقنية |
|--------|---------|
| **Frontend** | HTML5, CSS3 (Glass Morphism), Vanilla JavaScript |
| **Backend** | Supabase (PostgreSQL + Auth + Storage) |
| **Hosting** | Cloudflare Pages |
| **Fonts** | Tajawal (عربي), Cormorant (إنجليزي) |
| **Icons** | Emoji Native |
| **Maps** | Google Maps API (اختياري) |

## 📦 التثبيت والإعداد

### 1. استنساخ المشروع
```bash
git clone https://github.com/YOUR_USERNAME/dolcera.git
cd dolcera
```

### 2. إعداد المتغيرات البيئية
```bash
cp .env.example .env
# عدّل .env بمعلومات Supabase الخاصة بك
```

### 3. إعداد قاعدة البيانات
راجع [DEPLOYMENT.md](./DEPLOYMENT.md) للسكريبتات الكاملة لإنشاء الجداول.

### 4. النشر على Cloudflare Pages
1. ادفع الكود إلى GitHub
2. اربط المستودع بـ Cloudflare Pages
3. أضف المتغيرات البيئية
4. انشر!
